## Array and list
### What is array - display or arrange (things) in a particular way.

1. Basic array declaration
```javascript

    let setOfNumbers = [1,2,3,4,5]
    //always use [element,]
```

2. store more meaningfull data inside array
```javascript

    let student = [
        {
            name: "joe",
            score: 2000,
        },
        {
            name: "merry jane",
            score: 20,
        }
    ];


```
