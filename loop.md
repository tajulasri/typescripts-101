##Loop
### what is loop - a structure, series, or process, the end of which is connected to the beginning.


1. Basic loop
```javascript

   for(var i = 0; i < 10; i++) {
       console.log("we are at - " + i);
   }
```

2. Loop through array and list
```javascript

    let arrayOfNumbers = [2,3,4,5,6]

    for(let i in arrayOfNumbers) {
        console.log(i);
    }
```
