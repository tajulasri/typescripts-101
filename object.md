## Object
## what is Object ?

1. Basic declaration of object
```javascript 
    
    var person = {

        name: "john",
        age: 23,
        position: "software engineer"
    }

    console.log(person.name + " position: " + person.position)
```

### Other way of declaration

```javascript 
    
   let person = {
        name: "john",
        age: 23,
        position: "software engineer"
    }

    console.log(person.name + " position: " + person.position)

```
